# GitHub PR Cleaner

A Chrome extension that automatically hides "deployed to" messages from the GitHub PR timeline so you can focus on other things. This is 100% AI generated, no guarantees.

## Installation (Developer Mode)

1. Clone or download this folder:

   ```bash
   git clone https://github.com/mguida22/github-pr-cleaner.git
   ```

2. Open **Chrome** and navigate to:

   ```
   chrome://extensions
   ```

3. Enable **Developer mode** (toggle at top right).

4. Click **Load unpacked** and select your folder.

5. Open or refresh a GitHub Pull Request page â€” the "deployed to" comments will now be hidden automatically.
